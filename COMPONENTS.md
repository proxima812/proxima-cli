# Proxima UI — Components

## Commands

```bash
# Инициализация проекта (один раз)
bunx ui_proxima@latest init

# Добавить компонент
bunx ui_proxima@latest add <component-name>

# Удалить все изменения Proxima UI
bunx ui_proxima@latest reset
```

---

## Available components

### `socials`

Social links block with reusable icon buttons.

```bash
bunx ui_proxima@latest add socials
```

**Adds:**
- `src/components/social/Socials.astro`
- `src/components/social/SocialButton.astro`
- `src/data/social-icons.ts`
